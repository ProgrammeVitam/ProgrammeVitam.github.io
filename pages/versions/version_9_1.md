### Le code développé

<a name="901"></a>
### La version 9.0.1 - juin 2026

#### Les packages java (.jar) associés, diffusés pour Maven

Ils sont accessibles en suivant utilisant ce [lien](https://download.programmevitam.fr/vitam_repository/9.0.1/mvn_repo/).

#### Les paquets de déploiement

Ils permettent d'installer de manière outillée la solution, sous forme RPM pour Alma Linux et sous forme deb pour Debian. Ils contiennent aussi des conteneurs de documentation et de jeux de tests.  
    - Pour [Vitam](https://github.com/ProgrammeVitam/deployment/tree/9.0.1/vitam)  
    - Pour [Vitam UI](https://github.com/ProgrammeVitam/deployment/tree/9.0.1/vitam-ui)

##### *Vitam back-office*

- Publication [GitHub Vitam](https://github.com/ProgrammeVitam/vitam/tree/9.0.1)
- URL de configuration du [repository Maven](https://download.programmevitam.fr/vitam_repository/9.0.1/mvn_repo/) 
- URL de configuration des dépôts de binaires CentOS/RHEL :  
    - [vitam-product](https://download.programmevitam.fr/vitam_repository/9.0.1/rpm/vitam-product/)  
    - [vitam-external](https://download.programmevitam.fr/vitam_repository/9.0.1/rpm/vitam-external/)  
- URL de configuration des dépôts de binaires Debian :  
    - [vitam-product](https://download.programmevitam.fr/vitam_repository/9.0.1/deb/vitam-product/)  
    - [vitam-external](https://download.programmevitam.fr/vitam_repository/9.0.1/deb/vitam-external/)

##### *Vitam UI*

- Publication [GitHub Vitam UI](https://github.com/ProgrammeVitam/vitam-ui/tree/9.0.1)
- URL de configuration du [repository Maven](https://download.programmevitam.fr/vitamui_repository/9.0.1/mvn_repo/)
- URL de configuration des [dépôts de binaires CentOS/RHEL](https://download.programmevitam.fr/vitamui_repository/9.0.1/rpm/)
- URL de configuration des [dépôts de binaires Debian](https://download.programmevitam.fr/vitamui_repository/9.0.1/deb/)

Liens de téléchargement pour les partenaires :  [**version 9.0.1**](https://releases.programmevitam.fr/9.0.1/index.html)

#### La documentation fonctionnelle et technique

Elle est accessible, en partie sur GitHub pour les documents en forme brute RST, mais aussi directement sous forme PDF et/ou HTML sur la page [Documentation](/pages/documentation).
