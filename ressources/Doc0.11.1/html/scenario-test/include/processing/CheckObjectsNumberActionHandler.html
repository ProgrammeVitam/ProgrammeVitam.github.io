Cas de test OK¶

les 2 conditions suivantes doivent être respectées :

-nombre d’objets numériques déclarés dans le bordereau égal au nombre d’objets numériques référencés dans le bordereau.

-URI identique.

Cas de test KO¶

-présence doublons d’URI.

-nombre d’objets numériques déclarés dans le bordereau différent du nombre d’objets numériques référencés dans le bordereau.

-URI non identique
