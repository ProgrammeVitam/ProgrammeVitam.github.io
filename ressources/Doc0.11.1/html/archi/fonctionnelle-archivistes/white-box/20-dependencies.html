3.2.5. Dépendances logicielles¶

Voir aussi

La liste des dépendances logicielle exacte est décrite dans les releases notes de chaque version de VITAM.

3.2.5.1. Fournies¶

3.2.5.1.1. COTS¶

  • MongoDB : base de données orientée documents
  • Elasticsearch (+ plugins) : base d’indexation
  • Curator : maintenance des index d’elasticsearch
  • Logstash (+ plugins) : aggrégation et traitement des logs
  • Kibana : dashboards et recherche des logs techniques
  • Consul : annuaire de services
  • Siegfried : identification des formats de fichiers

3.2.5.1.2. Bibliothèques structurantes¶

  • Jetty : moteur de servlet

3.2.5.2. Requises¶

  • Java (JRE) 8

Voir aussi

Pour chaque version du système VITAM :

  • les composant fournis ou installés par dépendance sont précisés dans la documentation d’installation (DIN) ;
  • la liste des bibliothèques opensources incluses est précisée dans les release notes.
