Introduction
############

Objectif de ce document
=======================

Ce document a pour but de fournir à une équipe d'exploitants de :term:`VITAM` les procédures et informations utiles pour réaliser les montées de version de la solution logicielle. 

Ce document décrit les chemins de montées de versions supportés. 

Il s'adresse aux personnes suivantes :

* Les architectes techniques des projets désirant intégrer la solution logicielle VITAM ;
* Les exploitants devant installer la solution logicielle VITAM.
