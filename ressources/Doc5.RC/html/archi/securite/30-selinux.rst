SELinux
#######

Vitam peut fonctionner avec SELinux en mode ``enforcing``
Le support de SELinux a été pensé pour faire fonctionner tous les services utilisés par Vitam dans des domaines confinés (mode de sécurité le plus élevé).
Le fonctionnement des services Vitam dans le domaine non confiné n'a jamais été testé.
